# Ontologies

This directory contains the checked-in ontology sources used by the experiments.

- `family.ttl`: family-tree ontology used by Exp1 and Exp2 generator configs
- `family_tree.asp`: family-tree ASP program used by `gen-ft-asp`
- `toy.ttl`: smaller family ontology for toy checks and visual inspection
- `countries.ttl`: countries ontology used for ontology reasoning experiments
- `UNIV-BENCH-OWL2RL.owl`: OWL2Bench / Exp3 ontology source

Generated data should stay under `data/` and is ignored by Git.
